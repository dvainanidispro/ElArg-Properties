# Εφαρμογή Διαχείρισης Ακινήτων Δήμου

## Συνοπτική Παρουσίαση / Value Proposition

Ενιαία πλατφόρμα διαχείρισης δημοτικής ακίνητης περιουσίας και σχολικών κυλικείων, που αυτοματοποιεί διαδικασίες, παρέχει έγκαιρες ειδοποιήσεις, συγκεντρωτικά στατιστικά και ασφαλή πρόσβαση με ρόλους. Μειώνει τον διοικητικό φόρτο, αυξάνει την ακρίβεια των δεδομένων και διευκολύνει τη λήψη τεκμηριωμένων αποφάσεων.

## Οφέλη για Δήμους

- Εξάλειψη χειρωνακτικής διαχείρισης και σφαλμάτων από spreadsheets.
- Έγκαιρη συμμόρφωση με προθεσμίες και υποχρεώσεις.
- Ολοκληρωμένη εικόνα για τη μισθωτική και λειτουργική κατάσταση της περιουσίας.
- Διαφάνεια και πλήρης ιχνηλασιμότητα ενεργειών.
- Ευκολία κλιμάκωσης, προσαρμογής και προσθήκης νέων λειτουργιών.

## Κύριες Ενότητες Λειτουργικότητας

### 1. Διαχείριση Ακινήτων

- Καταγραφή όλων των ακινήτων (ιδιόκτητα, εκμισθωμένα, μισθωμένα, παραχωρημένα).
- Πλήρη πεδία: ΚΑΕΚ, ΑΤΑΚ, διεύθυνση, δημοτικό διαμέρισμα, χρήση, εμβαδά (οικοπέδου, κτίσματος, διαμερίσματος), έτος κατασκευής, ιδιοκτησιακό καθεστώς, άδειες, κ.λπ.
- Δυναμική αναζήτηση, φιλτράρισμα και ταξινόμηση
- Εξαγωγή λιστών σε Excel για διοικητική ή οικονομική χρήση

### 2. Πλήρες Αρχείο Μισθώσεων

- Κεντρική διαχείριση μισθώσεων ακινήτων και σχολικών κυλικείων.
- Πλήρη στοιχεία συμβάσεων: συμβαλλόμενοι, ημερομηνίες έναρξης–λήξης, μίσθωμα, συχνότητα πληρωμής, αριθμός δόσεων, κ.λπ.
- Ιστορικό αναπροσαρμογών μισθωμάτων
- Ιστορικό μισθώσεων ανά ακίνητο
- Εξαγωγή καρτέλας μίσθωσης σε Excel
- Ειδοποιήσεις για μισθώσεις που λήγουν σύντομα
- Ειδοποιήσεις για τρέχουσες αναπροσαρμογές μισθώσεων

### 3. Διαχείριση Συμβαλλομένων

- Ενιαίο μητρώο μισθωτών, εκμισθωτών και συνιδιοκτητών.
- Αναζήτηση με ΑΦΜ και στοιχεία επικοινωνίας.

### 4. Σχολικά Κυλικεία

- Αυτόματος υπολογισμός μισθώματος σχολικών κυλικείων με βάση τις μεταβλητές περιόδου (τριμήνου)
- Υποβολές στοιχείων από Διευθυντές μέσω ασφαλούς συνδέσμου (magic link)
- Αυτόματες επαναλαμβανόμενες υπενθυμίσεις μέσω email προς όσους δεν υπέβαλαν 
- Στατιστικά προόδου (ποσοστό υποβολών, εκκρεμότητες) και εξαγωγή σε Microsoft Excel για προώθηση σε οικονομική υπηρεσία
- Portal Διευθυντών Σχολείων με πρόσβαση μόνο στα δικά τους κυλικεία 

### 5. Χρήστες και Πρόσβαση

- Ασφαλής έλεγχος πρόσβασης ανά ρόλο (Admin, user, viewer, principal)
- Είσοδος με magic link ή παραδοσιακό login (username/password)
- Ελεγχόμενη είσοδος Διευθυντών Σχολείων αποκλειστικά για υποβολές στοιχείων.

### 6. Dashboard & Ειδοποιήσεις

- Κεντρικός πίνακας με KPIs: λήξεις μισθώσεων, στατιστικά υποβολών.
- Άμεση προβολή επερχόμενων αναπροσαρμογών μισθωμάτων 
- Άμεση προβολή επερχόμενων λήξεων μισθώσεων
